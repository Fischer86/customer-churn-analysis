## 🎯 Project Objective
This Exploratory Data Analysis (EDA) project aims to identify the key factors driving telecommunications customers to cancel their subscriptions (*Churn*). 

The business objective is not only to observe customer attrition but to uncover mathematical and visual correlations that will ultimately feed into a predictive classification model (Machine Learning).
